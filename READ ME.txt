Foram feitos comentários no arquivo index.html no github. A grafia em cinza claro no arquivo são os comentários referentes as funcionalidades do código.

Organizei os dados em pastas específicas de arquivos e linguagens.
Os dados preenchidos provavelmente estão indo via URL pelo método post.
Usado a TAG Required para deixar os campos obrigatórios.

Validação do CEP com preenchimento automático dos campos. (fonte: https://viacep.com.br/exemplo/javascript/).

Validação do e-mail com formato de e-mail (@).

Validação dos dados enviados com 2 alertas. O primeiro alerta avisa que o usuário deve preencher todos os campos obrigatórios. O segundo alerta avisa após o preenchimento e envio das informações a URL que os dados foram enviados com sucesso.

Usei como fonte para Css, Html e Javascript alguns vídeos (youtube) e sites (w3schools.com,https://www.horadecodar.com.br, além do conteúdo disponibilizado pela Gama Academy) na plataforma do curso.
